A Definition of Classes

[Syntax 1] Variables

Hypotheses

var x \mathrm{var} \ x var x

Assertion

term x \mathrm{term} \ x term x

Remarks

  1. var x \mathrm{var} \ x var x is a statement that means "x x x is a variable."
  2. term x \mathrm{term} \ x term x is a statement that means "x x x is a term."
  3. This syntax means that a variable denotes a term.

[Syntax 2] Sets

Hypotheses

term a \mathrm{term} \ a term a

Assertion

wff set a \mathrm{wff} \ \mathrm{set} \ a wff set a

Remarks

  1. wff φ \mathrm{wff} \ \varphi wff φ is a statement that means "φ \varphi φ is a wff (well-formed formula)."
  2. This syntax means that if a a a is a term, then set a \mathrm{set} \ a set a is a wff.
  3. set a \mathrm{set} \ a set a is a wff that means "a a a is a set."

[Syntax 3] Classes

Hypotheses

term A \mathrm{term} \ A term A

Assertion

wff class A \mathrm{wff} \ \mathrm{class} \ A wff class A

Remarks

  1. This syntax means that if A A A is a term, then class A \mathrm{class} \ A class A is a wff.
  2. class A \mathrm{class} \ A class A is a wff that means "A A A is a class."

[Syntax 4.1] Class Abstraction (1)

Hypotheses

  1. wff φ \mathrm{wff} \ \varphi wff φ
  2. var x \mathrm{var} \ x var x

Assertion

term {x | φ} \mathrm{term} \ \left \{ x \ \middle| \ \varphi \right \} term {x ∣ φ}

Remarks

This syntax means that a class abstraction is a term.

[Definition 4.2] Class Abstraction (2)

Hypotheses

  1. wff φ \mathrm{wff} \ \varphi wff φ
  2. var x \mathrm{var} \ x var x

Assertion

class {x | φ} \mathrm{class} \ \left \{ x \ \middle| \ \varphi \right \} class {x ∣ φ}

Remarks

This definition means that a class abstraction is a class.

[Definition 4.3] Class Abstraction (3)

Hypotheses

  1. wff φ \mathrm{wff} \ \varphi wff φ
  2. var x y \mathrm{var} \ x \ y var x y

Assertion

∀x[class x→(x∈{y | φ}↔set x∧[x/y]φ)] \forall x \left [ \mathrm{class} \ x \rightarrow \left ( x \in \left \{ y \ \middle| \ \varphi \right \} \leftrightarrow \mathrm{set} \ x \wedge \left [x / y \right] \varphi \right ) \right ] ∀x[class x→(x∈{y ∣ φ}↔set x∧[x/y]φ)]

Remarks

  1. var x y \mathrm{var} \ x \ y var x y is a statement that means "x x x and y y y are variables."
  2. [x/y]φ \left [x / y \right] \varphi [x/y]φ is the wff that results from the proper substitution of y y y for x x x in the wff φ \varphi φ. That is, y y y properly replaces x x x. (Megill 1993)

[Syntax 5.1] The Russell Class (1)

Assertion

term Ru \mathrm{term} \ \mathrm{Ru} term Ru

Remarks

This syntax means that the Russell class is a term.

[Definition 5.2] The Russell Class (2)

Hypotheses

var y \mathrm{var} \ y var y

Assertion

Ru={y | y∉y} \mathrm{Ru} = \left \{ y \ \middle| \ y \notin y \right \} Ru={y ∣ y∈/y}

Remarks

The Russell class is the class of all sets that are not members of themselves.

[Lemma 5.3] The Russell Class (3)

Assertion

class Ru \mathrm{class} \ \mathrm{Ru} class Ru

Proof

  1. var y \mathrm{var} \ y var y (Metavariable-type hypothesis)
  2. wff y∉y \mathrm{wff} \ y \notin y wff y∈/y (1, The syntax of axiomatic set theory)
  3. class {y | y∉y} \mathrm{class} \ \left \{ y \ \middle| \ y \notin y \right \} class {y ∣ y∈/y} (2, 1, Definition 4.2)
  4. Ru={y | y∉y} \mathrm{Ru} = \left \{ y \ \middle| \ y \notin y \right \} Ru={y ∣ y∈/y} (1, Definition 5.2)
  5. class Ru \mathrm{class} \ \mathrm{Ru} class Ru (4, 3, = = =-rewriting)

Remarks

The Russell class is a class.

[Theorem 6] Russell's Paradox

Assertion

Ru∉Ru∧¬set Ru \mathrm{Ru} \notin \mathrm{Ru} \wedge \neg \mathrm{set} \ \mathrm{Ru} Ru∈/Ru∧¬set Ru

Proof

  1. var x y \mathrm{var} \ x \ y var x y (Metavariable-type hypothesis)
  2. wff y∉y \mathrm{wff} \ y \notin y wff y∈/y (1, The syntax of axiomatic set theory)
  3. ∀x[class x→(x∈{y | y∉y}↔set x∧[x/y]y∉y)] \forall x \left [ \mathrm{class} \ x \rightarrow \left ( x \in \left \{ y \ \middle| \ y \notin y \right \} \leftrightarrow \mathrm{set} \ x \wedge \left [x / y \right] y \notin y \right ) \right ] ∀x[class x→(x∈{y ∣ y∈/y}↔set x∧[x/y]y∈/y)] (2, 1, Definition 4.3)
  4. ∀x[class x→(x∈{y | y∉y}↔set x∧x∉x)] \forall x \left [\mathrm{class} \ x \rightarrow \left ( x \in \left \{ y \ \middle| \ y \notin y \right \} \leftrightarrow \mathrm{set} \ x \wedge x \notin x \right ) \right] ∀x[class x→(x∈{y ∣ y∈/y}↔set x∧x∈/x)] (3, Proper substitution)
  5. term Ru \mathrm{term} \ \mathrm{Ru} term Ru (Syntax 5.1)
  6. class Ru→(Ru∈{y | y∉y}↔set Ru∧Ru∉Ru) \mathrm{class} \ \mathrm{Ru} \rightarrow \left ( \mathrm{Ru} \in \left \{ y \ \middle| \ y \notin y \right \} \leftrightarrow \mathrm{set} \ \mathrm{Ru} \wedge \mathrm{Ru} \notin \mathrm{Ru} \right ) class Ru→(Ru∈{y ∣ y∈/y}↔set Ru∧Ru∈/Ru) (5, 4, ∀ \forall ∀-elimination)
  7. Ru={y | y∉y} \mathrm{Ru} = \left \{ y \ \middle| \ y \notin y \right \} Ru={y ∣ y∈/y} (1, Definition 5.2)
  8. class Ru→(Ru∈Ru↔set Ru∧Ru∉Ru) \mathrm{class} \ \mathrm{Ru} \rightarrow \left ( \mathrm{Ru} \in \mathrm{Ru} \leftrightarrow \mathrm{set} \ \mathrm{Ru} \wedge \mathrm{Ru} \notin \mathrm{Ru} \right ) class Ru→(Ru∈Ru↔set Ru∧Ru∈/Ru) (7, 6, = = =-rewriting)
  9. class Ru \mathrm{class} \ \mathrm{Ru} class Ru (Lemma 5.3)
  10. Ru∈Ru↔set Ru∧Ru∉Ru \mathrm{Ru} \in \mathrm{Ru} \leftrightarrow \mathrm{set} \ \mathrm{Ru} \wedge \mathrm{Ru} \notin \mathrm{Ru} Ru∈Ru↔set Ru∧Ru∈/Ru (9, 8, Modus ponens)
  11. Ru∉Ru∧¬set Ru \mathrm{Ru} \notin \mathrm{Ru} \wedge \neg \mathrm{set} \ \mathrm{Ru} Ru∈/Ru∧¬set Ru (10, Definitions of conjunction and logical equivalence)

Remarks

  1. Let φ \varphi φ and ψ \psi ψ be wffs. If φ↔ψ∧¬φ \varphi \leftrightarrow \psi \wedge \neg \varphi φ↔ψ∧¬φ is true, then ¬φ∧¬ψ \neg \varphi \wedge \neg \psi ¬φ∧¬ψ is also true. This is a result of the definitions of conjunction and logical equivalence, and is used in Step 11 of the above proof.
  2. Since the Russell class is not a set, it is not an element of any classes. Therefore, the Russell class is not an element of itself.
  3. A proper class is a class that is not a set. The Russell class is a proper class.
  4. All sets are classes. We omit the proof here.
  5. Cf. Takeuti and Zaring 1982, chap. 4.
