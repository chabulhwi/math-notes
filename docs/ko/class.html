부류의 정의

[구문 1] 변수

가정

var x \mathrm{var} \ x var x

주장

term x \mathrm{term} \ x term x

붙임

  1. var x \mathrm{var} \ x var x는 "x x x는 변수이다."라는 뜻의 진술이다.
  2. term x \mathrm{term} \ x term x는 "x x x는 항이다."라는 뜻의 진술이다.
  3. 이 구문은 변수가 항을 나타냄을 뜻한다.

[구문 2] 집합

가정

term a \mathrm{term} \ a term a

주장

wff set a \mathrm{wff} \ \mathrm{set} \ a wff set a

붙임

  1. wff φ \mathrm{wff} \ \varphi wff φ는 "φ \varphi φ는 잘세운식[well-formed formula, wff]이다."라는 뜻의 진술이다.
  2. 이 구문은 a a a가 항이면 set a \mathrm{set} \ a set a가 잘세운식임을 뜻한다.
  3. set a \mathrm{set} \ a set a는 "a a a는 집합이다."라는 뜻의 잘세운식이다.

[구문 3] 부류

가정

term A \mathrm{term} \ A term A

주장

wff class A \mathrm{wff} \ \mathrm{class} \ A wff class A

붙임

  1. 이 구문은 A A A가 항이면 class A \mathrm{class} \ A class A가 잘세운식임을 뜻한다.
  2. class A \mathrm{class} \ A class A는 "A A A는 부류[class]이다."라는 뜻의 잘세운식이다.

[구문 4.1] 부류 추상화 (1)

가정

  1. wff φ \mathrm{wff} \ \varphi wff φ
  2. var x \mathrm{var} \ x var x

주장

term {x | φ} \mathrm{term} \ \left \{ x \ \middle| \ \varphi \right \} term {x ∣ φ}

붙임

이 구문은 부류 추상화[class abstraction]가 항임을 뜻한다.

[정의 4.2] 부류 추상화 (2)

가정

  1. wff φ \mathrm{wff} \ \varphi wff φ
  2. var x \mathrm{var} \ x var x

주장

class {x | φ} \mathrm{class} \ \left \{ x \ \middle| \ \varphi \right \} class {x ∣ φ}

붙임

이 정의는 부류 추상화가 부류임을 뜻한다.

[정의 4.3] 부류 추상화 (3)

가정

  1. wff φ \mathrm{wff} \ \varphi wff φ
  2. var x y \mathrm{var} \ x \ y var x y

주장

∀x[class x→(x∈{y | φ}↔set x∧[x/y]φ)] \forall x \left [ \mathrm{class} \ x \rightarrow \left ( x \in \left \{ y \ \middle| \ \varphi \right \} \leftrightarrow \mathrm{set} \ x \wedge \left [ x / y \right ] \varphi \right ) \right ] ∀x[class x→(x∈{y ∣ φ}↔set x∧[x/y]φ)]

붙임

  1. var x y \mathrm{var} \ x \ y var x y는 "x x x와 y y y는 변수이다."라는 뜻의 진술이다.
  2. [x/y]φ \left [ x / y \right ] \varphi [x/y]φ는 잘세운식 φ \varphi φ에 있는 변수 y y y를 변수 x x x로 정확히 바꿔 얻은 잘세운식이다. 이런 치환을 정확한 치환[proper substitution]이라고 한다. (Megill 1993)

[구문 5.1] 러셀 부류 (1)

주장

term Ru \mathrm{term} \ \mathrm{Ru} term Ru

붙임

이 구문은 러셀 부류가 항임을 뜻한다.

[정의 5.2] 러셀 부류 (2)

가정

var y \mathrm{var} \ y var y

주장

Ru={y | y∉y} \mathrm{Ru} = \left \{ y \ \middle| \ y \notin y \right \} Ru={y ∣ y∈/y}

붙임

러셀 부류란, 자신을 원소로 갖지 않는 모든 집합의 부류이다.

[따름 정리 5.3] 러셀 부류 (3)

주장

class Ru \mathrm{class} \ \mathrm{Ru} class Ru

증명

  1. var y \mathrm{var} \ y var y (메타변수 유형 가정)
  2. wff y∉y \mathrm{wff} \ y \notin y wff y∈/y (1, 공리적 집합론의 구문)
  3. class {y | y∉y} \mathrm{class} \ \left \{ y \ \middle| \ y \notin y \right \} class {y ∣ y∈/y} (2, 1, 정의 4.2)
  4. Ru={y | y∉y} \mathrm{Ru} = \left \{ y \ \middle| \ y \notin y \right \} Ru={y ∣ y∈/y} (1, 정의 5.2)
  5. class Ru \mathrm{class} \ \mathrm{Ru} class Ru (4, 3, = = = 다시 쓰기)

붙임

러셀 부류는 부류이다.

[정리 6] 러셀의 역설

주장

Ru∉Ru∧¬set Ru \mathrm{Ru} \notin \mathrm{Ru} \wedge \neg \mathrm{set} \ \mathrm{Ru} Ru∈/Ru∧¬set Ru

증명

  1. var x y \mathrm{var} \ x \ y var x y (메타변수 유형 가정)
  2. wff y∉y \mathrm{wff} \ y \notin y wff y∈/y (1, 공리적 집합론의 구문)
  3. ∀x[class x→(x∈{y | y∉y}↔set x∧[x/y]y∉y)] \forall x \left [ \mathrm{class} \ x \rightarrow \left ( x \in \left \{ y \ \middle| \ y \notin y \right \} \leftrightarrow \mathrm{set} \ x \wedge \left [ x / y \right ] y \notin y \right ) \right ] ∀x[class x→(x∈{y ∣ y∈/y}↔set x∧[x/y]y∈/y)] (2, 1, 정의 4.3)
  4. ∀x[class x→(x∈{y | y∉y}↔set x∧x∉x)] \forall x \left [ \mathrm{class} \ x \rightarrow \left ( x \in \left \{ y \ \middle| \ y \notin y \right \} \leftrightarrow \mathrm{set} \ x \wedge x \notin x \right ) \right ] ∀x[class x→(x∈{y ∣ y∈/y}↔set x∧x∈/x)] (3, 정확한 치환)
  5. term Ru \mathrm{term} \ \mathrm{Ru} term Ru (구문 5.1)
  6. class Ru→(Ru∈{y | y∉y}↔set Ru∧Ru∉Ru) \mathrm{class} \ \mathrm{Ru} \rightarrow \left ( \mathrm{Ru} \in \left \{ y \ \middle| \ y \notin y \right \} \leftrightarrow \mathrm{set} \ \mathrm{Ru} \wedge \mathrm{Ru} \notin \mathrm{Ru} \right ) class Ru→(Ru∈{y ∣ y∈/y}↔set Ru∧Ru∈/Ru) (5, 4, ∀ \forall ∀ 제거)
  7. Ru={y | y∉y} \mathrm{Ru} = \left \{ y \ \middle| \ y \notin y \right \} Ru={y ∣ y∈/y} (1, 정의 5.2)
  8. class Ru→(Ru∈Ru↔set Ru∧Ru∉Ru) \mathrm{class} \ \mathrm{Ru} \rightarrow \left ( \mathrm{Ru} \in \mathrm{Ru} \leftrightarrow \mathrm{set} \ \mathrm{Ru} \wedge \mathrm{Ru} \notin \mathrm{Ru} \right ) class Ru→(Ru∈Ru↔set Ru∧Ru∈/Ru) (7, 6, = = = 다시 쓰기)
  9. class Ru \mathrm{class} \ \mathrm{Ru} class Ru (따름 정리 5.3)
  10. Ru∈Ru↔set Ru∧Ru∉Ru \mathrm{Ru} \in \mathrm{Ru} \leftrightarrow \mathrm{set} \ \mathrm{Ru} \wedge \mathrm{Ru} \notin \mathrm{Ru} Ru∈Ru↔set Ru∧Ru∈/Ru (9, 8, 전건 긍정 논법)
  11. Ru∉Ru∧¬set Ru \mathrm{Ru} \notin \mathrm{Ru} \wedge \neg \mathrm{set} \ \mathrm{Ru} Ru∈/Ru∧¬set Ru (10, 논리곱과 쌍조건문의 정의)

붙임

  1. φ \varphi φ와 ψ \psi ψ가 잘세운식이라고 하자. φ↔ψ∧¬φ \varphi \leftrightarrow \psi \wedge \neg \varphi φ↔ψ∧¬φ가 참이면 ¬φ∧¬ψ \neg \varphi \wedge \neg \psi ¬φ∧¬ψ 또한 참이다. 이는 논리곱과 쌍조건문의 정의에 따른 결과이며, 위의 증명의 11단계에서 이용됐다.
  2. 러셀 부류는 집합이 아니므로 어떤 부류의 원소도 될 수 없다. 따라서 러셀 부류는 그 자신의 원소가 아니다.
  3. 집합이 아닌 부류를 진부류[proper class]라고 한다. 러셀 부류는 진부류이다.
  4. 모든 집합은 부류이다. 여기에서는 그 증명을 생략하겠다.
  5. Takeuti and Zaring 1982, chap. 4를 참고하라.
